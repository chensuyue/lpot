Tutorial Examples¶

These examples are easy to run without preparing dataset. The script will download public dataset automatically or create fake dataset.

  1. Compare existed FP32 & INT8 ResNet50 model directly: Performance of FP32 Vs. INT8 ResNet50 Model.

  2. An End-To-End pipeline to build up a CNN model by Pytorch to recognize fashion image and speed up AI model by Intel® Neural Compressor: Intel® Neural Compressor Sample for PyTorch*

  3. An End-To-End pipeline to build up a CNN model by Tensorflow to recognize handwriting number and speed up AI model by Intel® Neural Compressor: Intel® Neural Compressor Sample for TensorFlow*
